## 📋 О проекте

**Worker Selection App** — это AI-платформа, призванная полностью автоматизировать работу HR-отдела на этапе первичного отбора. В своей финальной версии система будет самостоятельно совершать **автоматический обзвон** баз кандидатов, проводить интервью, анализировать ответы и отбирать наиболее перспективных сотрудников (станочников, сварщиков, техников).

Ключевая особенность — не просто проверка навыков, а **прогнозирование удержания (Retention Prediction)**. Система анализирует совместимость бытовых условий кандидата с требованиями завода, отсеивая тех, кто уволится в первые 3 месяца.

### 🎯 Ключевые возможности

#### ✅ Реализовано в MVP (Текущая версия):
- **🧠 Локальная LLM** — использование модели **Qwen 3 (4B)** для извлечения фактов и структурирования данных (работает offline).
- **📈 ML-прогнозирование** — модель **CatBoost** оценивает риск увольнения на основе синтетических данных, сгенерированных по правилам предметной области.
- **⚙️ Полная автономность** — при первом запуске система сама генерирует датасет, обучает модель и поднимает UI.
- **🔍 Интерпретируемость** — объяснение причин риска (например, "Дорога > 90 мин", "Нет сертификатов").

#### ⏳ В планах (Целевая версия):
- **📞 AI-Телефония** — интеграция с VoIP для автоматического обзвона и проведения голосовых интервью без участия человека.
- **☁️ Гибридный AI** — возможность выбора между локальной LLM (для приватности) и облачными API (для скорости).
- **📊 Продвинутая ML-аналитика** — расширение вектора признаков и усложнение модели удержания для повышения точности.

## 🚀 Быстрый старт

### Требования

- Python 3.10+
- **GPU** (рекомендуется для LLM Qwen-4B) или 16GB+ RAM для работы на CPU.
- CUDA (если используется GPU).

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository_url>
   cd genai-project
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Запуск

Система запускается одной командой. Скрипт `main.py` автоматически:
1. Инициализирует базу данных.
2. Генерирует обучающий датасет (если его нет).
3. Обучает ML-модель (CatBoost).
4. Загружает LLM в память.
5. Запускает API (FastAPI) и Frontend (Streamlit).

```bash
python main.py
```

После запуска доступны:
- **API Docs**: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
- **UI Dashboard**: [http://localhost:8501](http://localhost:8501)

## 🏗️ Архитектура системы (MVP)

Система работает как монолит с разделением процессов:

1. **Initialization**: Генерация `train_dataset.csv` (1000 строк) → Обучение `CatBoostClassifier`.
2. **Processing Pipeline**:
   **Input Audio** → **STT** → **LLM (Qwen 4B)** → **Normalization** → **ML Prediction** → **UI Display**

Компоненты взаимодействуют внутри одного runtime, при этом Streamlit запускается как subprocess.

## 🎭 Демо-сценарии

В UI предустановлены два пресета для демонстрации работы ML-ядра:

### 🟢 "Идеальный кандидат" (Green Zone)
- **Профиль**: Опытный (10 лет), живет близко (20 мин), дневная смена.
- **Результат**: Retention Score **> 75%**.
- **Вердикт**: ✅ Рекомендуется к найму.

### 🔴 "Проблемный кандидат" (Red Zone)
- **Профиль**: Мало опыта, долгая дорога (2 часа), требует высокую ЗП, только ночные смены.
- **Результат**: Retention Score < 30%.
- **Факторы риска**: "Дорога > 90 мин", "Мало навыков", "Несовместимость графика".
- **Вердикт**: ⛔ Высокий риск увольнения.

## 🚧 Планы развития

Проект движется в сторону полной автоматизации HR-процессов. Следующие шаги:

1. **Автоматический обзвон (Robocalling)**
   - Подключение SIP-телефонии.
   - Бот сам звонит кандидату, задает вопросы и записывает ответы.
   - Фильтрация "недозвонов" и автоответчиков.

2. **Гибкость в выборе LLM**
   - Реализация переключателя: использовать локальную модель (бесплатно, приватно) или внешний API (OpenAI/Anthropic/GigaChat) для повышения качества распознавания сложных диалогов.

3. **Эволюция ML-ядра**
   - Углубление модели предсказания удержания.
   - Расширение вектора признаков (добавление психотипов, истории предыдущих мест работы).
   - Повышение точности за счет сбора реальной статистики увольнений.
